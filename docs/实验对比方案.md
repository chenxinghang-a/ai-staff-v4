## 实验环境

## 答辩Demo顺序
1. 快速演示：简单电气概念查询 → 秒出（效率）
2. 能力展示：snap7 PLC读取 → Skill注入 → 高质量代码（专业性）
3. **反差演示（高潮）**：故意违反GB 50054 → 审核专家警告 → V5门控 → 自动迭代修正（可靠性）
4. 成本展示：不同任务Token消耗差异（工程实用性）
